## About Blocks

in Matscraft you got 5 types of ore blocks, each with its own rarity vibe:

- Common Mats Ore
- Uncommon Mats Ore
- Rare Mats Ore
- Epic Mats Ore
- Legendary Mats Ore

Each block drops a different amount of **$MATS**, depending on how rare it is.

<Note type="info">
  To get the $MATS drops, you’ll need a **special pickaxe** for each type of
  block. [Learn more about pickaxes](pickaxe-ability/)
</Note>
