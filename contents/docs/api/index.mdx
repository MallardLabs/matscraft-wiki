## Auth
- PATH: `/api/users/auth`
- METHOD: `POST`
- BODY:`JSON`
  - `minecraft_id`
  - `minecraft_username`
  - `token`

## Update Balance
- PATH: `/api/users/update-balance`
- METHOD: `PUT`
- BODY:`JSON`
- `discord_id`
- `amount`
- HEADERS: `matscraft-secret`

### Response
```json

{
    "balance": "999"
}

```
